Requires to have Java 1.7 or above to run the program.
Check and make sure that the port you use is not blocked by the firewall.

1) Build the source code by running "make".

From the command line,

2)

	a) To run TCP Server, input the following command :

		java TCPServer port#

	   To run TCP Client, input the following command :

		java TCPClient server_address server_port#

	b) To run UDP server, input the following command :

		java UDPServer port#

	   To run UDP client, input the following command :

		java UDPClient server_address server_port#
